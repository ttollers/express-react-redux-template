# express-react-redux-template
A basic expres react/redux app. Has examples of async redux actions and middleware.
No selectors for the time being.

## Usage
### Install
    npm install

or

    yarn

### Compile Front End
    npm run build

### Run
    npm start
With a watcher:

    npm run dev
    
With Docker:

    docker build -t my-image .
    
    docker run -d -p 8080:8080 my-image

http://localhost:8080
